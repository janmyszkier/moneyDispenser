# moneyDispenser

Retrieve unlimited money from console

#### Installation
```
composer install
```

#### Usage
```
php bin/console money:retrieve 180
```

Sample output:
```
$ php bin/console money:retrieve 180
Here is your 180:
[100 PLN, 50 PLN, 20 PLN, 10 PLN]

```

#### Testing
```
php vendor/bin/phpunit
```
